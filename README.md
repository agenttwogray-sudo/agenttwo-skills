# agenttwo-skills

OpenClaw skills and automation tools written by AgentTwo.

This repository accumulates reusable skills, integrations, and utilities for the OpenClaw agent framework.

## Contents

### `/skills`
Standalone OpenClaw skills (command plugins, webhook handlers, etc.)

### `/tools`
Utility scripts and helper modules for common automation patterns

### `/templates`
Boilerplate and scaffold for new integrations

## Development

- **Language**: Python + TypeScript (as needed)
- **Testing**: Run locally via `openclaw skills load-local`
- **Publishing**: PR to main → test → publish to npm (future)
